# ZapretGUI

Графический интерфейс для [zapret-discord-youtube](https://github.com/Flowseal/zapret-discord-youtube) — инструмента обхода блокировок Discord и YouTube на Windows.

Приложение не модифицирует и не заменяет сам `zapret` — оно предоставляет удобный интерфейс к уже существующим `.bat`-файлам сборки.

---

## Возможности

- **Управление стратегиями** — выбор профиля обхода из выпадающего списка, запуск и остановка в один клик
- **Управление службами** — установка, удаление и проверка статуса службы Windows, обновление файла hosts
- **Встроенная диагностика** — запуск диагностики zapret с цветным выводом прямо в журнале приложения
- **Тесты zapret** — запуск PowerShell-теста в реальном времени с анализом результатов по сайтам
- **Редактор конфигурации** — просмотр и редактирование пользовательских списков доменов без выхода из приложения
- **Обновление zapret** — автоматическая проверка версии при запуске, скачивание и установка обновления в один клик
- **Автозапуск с Windows** — через Планировщик задач, с правами администратора и без UAC-запроса
- **Системный трей** — сворачивание в трей при закрытии окна
- **Журнал операций** — цветная подсветка событий (успех / предупреждение / ошибка)
- **Светлая и тёмная темы** — переключение в настройках

---

## Требования

- **Windows 10 / 11** (64-bit)
- Установленный **[zapret-discord-youtube](https://github.com/Flowseal/zapret-discord-youtube)** (папка с `service.bat`)
- Права **администратора** (необходимы для управления службами и перехвата трафика)

## Установка и запуск

### Вариант 1 — готовый .exe (рекомендуется)

1. Скачайте последний релиз из раздела [Releases](../../releases)
2. Распакуйте архив в любую папку
3. Запустите `ZapretGUI.exe` — Windows один раз запросит подтверждение прав администратора
4. В разделе **Настройки → Путь к программе zapret** укажите папку с установленным zapret

> **Два варианта сборки:**
> - `onefile` — один файл `ZapretGUI.exe`, удобно переносить
> - `onedir` — папка с файлами, быстрее запускается (нет распаковки при старте)

### Вариант 2 — запуск из исходников

**Требования:** Python 3.8 или новее

```bat
:: 1. Установить зависимости Python
install_deps.bat

:: 2. Запустить приложение
run.bat
```

Или вручную через терминал:

```bat
pip install -r requirements.txt
python zapret_gui.py
```

## Первый запуск

1. Запустите `ZapretGUI.exe`
2. Откройте вкладку **Настройки**
3. Укажите **путь к папке zapret** (папка, в которой лежит `service.bat`) и нажмите **✓ Применить**
4. Вернитесь на вкладку **Управление**, выберите профиль из списка и нажмите **▶ Запустить**

## Описание вкладок

| Вкладка | Что делает |
|---|---|
| **Управление** | Выбор профиля (стратегии), кнопки «Запустить» / «Остановить», статус процесса |
| **Службы** | Установка и удаление службы Windows, статус, обновление hosts |
| **Конфигурация** | Редактирование списков доменов и конфиг-файлов с подсветкой |
| **Диагностика** | Запуск встроенной диагностики zapret с выводом в журнал |
| **Тесты** | Запуск `test zapret.ps1` с покзом результатов по каждому сайту |
| **Настройки** | Путь к zapret, тема оформления, автозапуск, обновление zapret, экспорт/импорт настроек |

## Обновление zapret

ZapretGUI умеет обновлять установленный zapret автоматически:

1. Перейдите в **Настройки → Обновление zapret**
2. Нажмите **🔍 Проверить обновление**
3. Если доступна новая версия — нажмите **⬇ Обновить zapret**

Приложение выполнит все шаги само:
- остановит службу `zapret` и процесс `winws.exe`
- скачает официальный `.zip`-архив с GitHub
- заменит файлы установки (пользовательские списки доменов **сохранятся**)
- перезапустит службу, если она работала до обновления

> При каждом запуске ZapretGUI тихо проверяет версию в фоне и сообщает только тогда, когда доступно обновление. Если версия актуальна — никаких всплывающих окон не будет.

## Автозапуск

Для запуска ZapretGUI вместе с Windows:

**Настройки → Автозапуск при старте Windows** → поставьте галочку.

Приложение создаст задачу в **Планировщике задач Windows** с уровнем привилегий «Наивысший доступный». Это единственный надёжный способ запускать приложение с правами администратора **без UAC-запроса** при каждом входе в систему.

---

**Требования:** Python 3.8+, доступ в интернет.

## Частые вопросы

**Антивирус ругается на ZapretGUI.exe**

Ложное срабатывание — типично для любых .exe, собранных через PyInstaller. Добавьте файл в исключения антивируса или соберите из исходников самоcтоятельно

**Стратегии не отображаются в списке**

Убедитесь, что в Настройках указан верный путь к папке zapret — там должны лежать файлы `general*.bat` и `service.bat`.

**Кнопки служб не реагируют или зависают**

Проверьте, что приложение запущено с правами администратора и путь к zapret задан корректно.

**UAC-запрос при каждом запуске**

Это нормальное поведение при ручном запуске .exe с манифестом администратора. Чтобы убрать UAC-запрос при старте системы — включите автозапуск через Планировщик задач в Настройках.
